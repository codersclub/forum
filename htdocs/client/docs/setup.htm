Установка


Нормальный инсталлятор появился только в версии 1.104. С ним все просто: заходим на в раздел Скачать программу, выбираем версию, скачиваем, запускаем, следуем стандартным инструкциям.

Есть только такой ньюанс: если ранее использовалась версия 1.003 или ниже, то инсталлятор не сможет определить каталог, в котором "живет" программа и предложит установку в Program Files. В этом случае можно либо указать правильный каталог, либо после установки переписать файл data.mdb из старого местожительства

Если есть желание установить более раннюю версию, то вот как это было:

Программа предназначена для работы с форумом. Установка программы не требуется (я сам очень не люблю всех этих дистрибутивов, серийных номеров и прочих премудростей). Необходимо просто скачать саморазворачивающийся архив и поместить его в любую папку, нажать "Enter" и дождаться окончания разархивации. После этого архив можно удалить. Все. Программа установлена. Пытаемся ее запустить. Под Windows 2000, Windows XP проблем никаких не должно быть. Если это не так, свяжитесь с разработчиком (см. п. "Контактная информация"). Под Windows 9х может быть потребуется дополнительно установить компонент "ActiveX Data Objects (ADO)". Руководство по установке ADO см. на сайте фирмы Microsoft (http://www.microsoft.com). В следующей версии будет подробное разъяснение (конечно же, если оно потребуется).
