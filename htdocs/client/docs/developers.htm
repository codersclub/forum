Информация для разработчиков, любителей совершенствования, а также хакеров


Программа состоит из серверной и клиентской частей. Серверная часть:

  • выполняет идентификацию пользователей средствами программной части форума;
  • общается с базой данных - хранилищем сообщений в форуме;
  • выводит результаты в определенном формате

Клиентская часть

  • принимает данные от серверной части;
  • трансформирует их в записи в локальной базе данных в формате MS Access;
  • предоставляет интерфейс просмотра и поиска сообщений по различным критериям

Клиентская часть программы реализована на Visual Studio 6.0 и ActiveXR Data Objects (ADO). В программу не встроено никаких механизмов защиты. Можно свободно обмениваться каталогом Data. Например, не зарегистрированные пользователи могут использовать эту возможность для доступа к сообщениям не из главного форума (при этом обновление по-прежнему будет регламентироваться серверной частью, поэтому доступ будет только к старым сообщениям). Локальная база данных тоже никак не защищена. Можно пользоваться всеми средствами MS Access, создавая, например, запросы, формы и отчеты. Автор будет приветствовать предложения по изменению базы. Хорошие предложения будут внедряться в последующие версии. Все обсуждаемо. Обсуждаем даже вариант публикации исходного кода клиентской части. В этом случае, опять же, возможны различные варианты предложений по усовершенствованию. Серверная часть реализована как Add-In к движку форума. Исходя из целей безопасности форума исходный код серверной части программы, вряд ли, будет когда-либо опубликован. Однако формат формируемых результатов может быть изменен. Также обсуждаема разработка дополнительных модулей для получения дополнительной информации из форума
